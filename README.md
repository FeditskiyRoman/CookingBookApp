# CookingBookApp

node.js should be installed

testdb is sql db, It should be created

run "npm i" in the cmd

# Development

    # Frontend
    Run in cmd:
        gulp dev

    # Backend
    Run in cmd:
        cd dev
        node index

# Production
    Run in cmd:
        gulp build
        node dev/index
    Open 'http://localhost:8081/'
